PIXDOCK

Pixel-art Docker dashboard with retro CRT aesthetics. Monitor and control containers with LED indicators, HP-bar gauges, and scanline effects.

One-line install
$ curl -fsSL https://raw.githubusercontent.com/stormixus/pixdock/main/install.sh | sh
HP
100%

// FEATURES

AUTO-DETECT
Swarm or Standalone mode, detected automatically. Zero configuration needed.
REAL-TIME
Shared WebSocket broadcast every 3s. Scales to many concurrent clients.
CONTROLS
Start, stop, restart containers. Scale Swarm services with one click.
SWARM SUPPORT
Node rack, services by stack, task details, and replica scaling.
AUTH
Optional bearer token auth via PIXDOCK_TOKEN. REST + WebSocket secured.
PIXEL UI
CRT scanlines, Press Start 2P font, LED indicators, HP-bar gauges.
CONTAINER LOGS
Built-in log viewer with search, filter, and auto-refresh.
TINY FOOTPRINT
Single binary, ~128MB memory limit, Alpine-based image. Air-gap ready.

// HOW IT WORKS

INSTALL
Run the one-liner. It pulls the image and sets up a compose file in ~/.pixdock/.
curl -fsSL ...install.sh | sh
CONFIGURE
Optionally set a port and auth token. Defaults work out of the box.
--port 9000 --token my-secret
MONITOR
Open localhost:8420 and see all your containers in glorious pixel art.
http://localhost:8420
